Pi-hole QEI

Pi-hole quick and easy install
PowerShell based scripts that generate a PiOS image with all necessary headless setup for Pi-hole. The script uses predefined arguments file with some default values. Technically it's as easy as running the script and waiting for image file to appear for later etching onto SD card.
Most of this project was prepared using shell scripts. Although there is a little of PowerShell.
