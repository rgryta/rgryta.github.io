NoPrint

Do not allow prints in your code.
Python package that can be used alongside all available linting or code styling packages. There are no blocking dependencies (unlike flake8-print) so you can use it without worries.
The entire project is one Python package. Repository contains CI/CD pipeline for quick package build checks and automatic release to PyPi when published. Local development environment setup is also automated using WSL2 scripts - for native `make` support.
