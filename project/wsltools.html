WSLTools

PowerShell Module WSLTools.
PowerShell Module with purpose of allowing easy and quick installation of different WSL distros straight from source - including not officially supported alpine. It contains several methods: Ensure HyperV is enabled Ensure WSL is installed Install Alpine Install Ubuntu Run after Reboot Install (new) PowerShell
The entire project is a single PowerShell Module. Repository contains CI/CD pipeline for quick deployment to PowerShell Gallery when release is published.
