Accelerator Design With OpenCL

Several quick programs allowing for applying video filters. Made for ATHENS Network student exchange.
Write basic OpenCL programs (both host program and kernel) for FPGAs Write basic OpenCL programs for programming GPUs Be familiar with notions of optimization for performance
Each program consists of three parts: Makefile, CPP and OpenCL sourcefiles. Programs are then uploaded onto the microcontrollers via SSH and run remotely.
