Verified Bird

Chrome extension to modify verification checkmarks.
This is official repository of Verified Bird - get rid of unnecessary "Verified" checkmark from Twitter in one click. If you wish to use a different emoji or text, so be it. Simply open extension page (click on extension icon) and customize it. You can also add verification checkmark to people who are not yet verified.
Extension made purely in HTML, CSS and JS.
