Barman

STM32-based cocktail-making machine for SST and MISK subjects.
Student project integrating complex embedded STM32 system with an Android application using Bluetooth protocol, allowing for creating cocktails and other beverages.
System was split into two parts. Microcontroller that was responsible for controlling pumps connected to the refill bottles and Android application that used Bluetooth to connect to the STM32 to send recipies and intstructions. Exchange of information was performed in duplex mode with the app constantly notifying the user about current process' progress.
