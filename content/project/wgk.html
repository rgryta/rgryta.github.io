Isometric Dungeon Crawler

 Unreal Engine 4 Project for WGK subject.
Abstract. 3D dungeon-crawler set in an isometric view done in Unreal Engine 4.25.4.0. Features: - AI allowing for area queries in search of a position that allows to shoot at player - Dynamic weapon switching (using the same event with different outcomes depending on equipped weapon) - Inverse kinematics for holding weapons (allows for a weapon to be in the same position all the time) - Multiple levels and menu with proper switching logic
Architecture. Whole project was done using Unreal Engine 4 (version 4.25.4.0). Technology: Unreal Engine 4 Links: GitHub Page
